# Storage System Monitor 🛠️

A lightweight Python utility designed to monitor storage device health and provide real-time status logging. 
## 📌 Features
- Monitors multiple storage devices simultaneously.
- Provides categorized health status: `Healthy`, `Warning`, and `Critical`.
- Zero-dependency script (uses standard Python libraries).
