Python Lists Code Challenge

Challenge: Lists Basics
This challenge tests your understanding of core Python list operations, including creating, accessing, modifying, and managing list items. 



Complete the following steps within the script:

Create a list called colors with the values: "red", "green", "blue".

Print the first item in the list.
Change the second item to "yellow".
Add "purple" to the end of the list using .append().
Remove "red" from the list using .remove().
Print the final list to the console. 




How to Run
Ensure you have Python installed.
Run the script using your terminal or IDE:
bash
python Lists.py
Use code with caution.
