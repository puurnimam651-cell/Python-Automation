# Python Variables Basics

A simple Python script demonstrating how to declare, use, and manipulate variables. This is part of my Python learning journey on **KodeKloud**.

## 📝 Description
This script covers the fundamental concepts of Python variables, including:
- **Data Types**: Strings, Integers, Floats, and Booleans.
- **F-Strings**: Modern way to format and print variables.
- **Basic Operations**: Performing math with numeric variables.
- **Dynamic Typing**: Showing how a variable can change from a number to text.
