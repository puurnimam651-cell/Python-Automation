# Python Operators Basics

A practical guide to using **Arithmetic**, **Comparison**, **Assignment**, and **Logical** operators in Python. 

## 📝 Overview
This script demonstrates how Python handles math and logic, including:
- **Arithmetic**: Basic math like addition (`+`), modulus (`%`), and exponents (`**`).
- **Comparison**: Checking values using equal to (`==`), not equal (`!=`), and greater than (`>`).
- **Assignment**: Shortcut ways to update variables (e.g., `+=`).
- **Logical**: Combining conditions using `and`, `or`, and `not`.
